# Student Management REST API

## 📌 Description

A REST API built using Spring Boot, Spring Data JPA, and MySQL to perform CRUD operations on student records.

---

## 🚀 Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

---

## ✨ Features

- Add Student
- View All Students
- Search Student by ID
- Update Student
- Delete Student

---

## 📂 Project Structure

controller/
entity/
repository/
service/

---

## 🛠 Database

Database Name:

```text
StudentAPI_db
```

---

## 📮 API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| POST | /students | Add Student |
| GET | /students | Get All Students |
| GET | /students/{id} | Get Student By ID |
| PUT | /students/{id} | Update Student |
| DELETE | /students/{id} | Delete Student |
